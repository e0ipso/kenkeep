# Project knowledge base

AI-session-derived knowledge for this repo, built by [`kenkeep`](https://github.com/e0ipso/kenkeep). Everything here is plain markdown.

- `nodes/` — the knowledge, by kind (`practice/`, `map/`). Reviewed via git like any other change.
- `ENTRY.md` / `GRAPH.md` — generated summaries, injected into or read by new AI sessions. Regenerated on commit.

Add a node with `/kk-add`, seed from existing docs with `/kk-bootstrap`, and process captured sessions with `/kk-curate`. See the [docs](https://github.com/e0ipso/kenkeep) for the full reference.
