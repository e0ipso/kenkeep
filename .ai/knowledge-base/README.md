# Project knowledge base

AI-session-derived knowledge for this repo, built by [`@e0ipso/ai-knowledge-base`](https://github.com/e0ipso/ai-knowledge-base). Everything here is plain markdown.

- `nodes/` — the knowledge, by kind (`practice/`, `map/`). Reviewed via git like any other change.
- `INDEX.md` / `GRAPH.md` — generated summaries, injected into or read by new AI sessions. Regenerated on commit.

Add a node with `/kb-add`, seed from existing docs with `/kb-bootstrap`, and process captured sessions with `/kb-curate`. See the [docs](https://github.com/e0ipso/ai-knowledge-base) for the full reference.
