# `GRAPH.md`

Deterministic edge listing built from every node's `relates_to` (loose) and `depends_on` (strict) frontmatter. Generated by `generateGraph` (`src/lib/index-gen.ts`), a pure function of `nodes/` plus an injected `now`.

Unlike `INDEX.md`, `GRAPH.md` is **not** injected into every session. The harness reads it on demand when it needs the full graph.

Carries `GraphFrontmatterSchema` frontmatter (same shape as `IndexFrontmatterSchema`: `schema_version`, `nodes_hash`, `node_count`).

Regeneration runs alongside `INDEX.md` at the end of every `curate` run and via `index rebuild`. The lint-staged pre-commit recipe (`index rebuild --stage`) keeps both files in lockstep with committed nodes.
