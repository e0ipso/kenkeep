The `templates/` directory and `.claude/hooks/*.mjs` files are build output, not hand-written source. `templates/` is fully regenerated by `scripts/build-templates.mjs` (which `rmSync`s the dest before rebuilding) from `src/templates-source/` plus `dist/hooks/`. `.claude/hooks/*.mjs` is this repo dogfooding its own `init`.

These should be gitignored and removed from the repo. Publishing remains safe because `prepublishOnly: npm run build` regenerates `templates/` before every npm publish, and `templates/` is listed in `package.json` `"files"`.
