# Core Concepts

A per-repo store of curated knowledge built from your AI sessions. Three pipelines:

- **Capture** turns raw transcripts into structured candidates (two stages).
- **Curate** turns candidates into proposals for human review.
- **Consume** injects the current `INDEX.md` at every session start.
